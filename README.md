## 📖 Deskripsi

Sistem Informasi Manajemen Parkir Tunai merupakan aplikasi berbasis **Command Line Interface (CLI)** yang dikembangkan menggunakan bahasa pemrograman **Python** sebagai proyek Ujian Akhir Semester Mata Kuliah **Dasar Pemrograman**.

Aplikasi ini dirancang untuk membantu proses pengelolaan parkir kendaraan secara sederhana, mulai dari pencatatan kendaraan masuk, kendaraan keluar, perhitungan biaya parkir, hingga penyusunan laporan parkir harian.

# 🚀 Cara Menjalankan

Clone repository

```bash
git clone https://github.com/username/repository.git
```

Masuk ke folder project

```bash
cd repository
```

Install PrettyTable

```bash
pip install prettytable
```

Jalankan program

```bash
python main.py
```
