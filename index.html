Sistem Informasi Manajemen Parkir Tunai

Aplikasi berbasis Command Line Interface menggunakan Python

Project Ujian Akhir Semester - Dasar Pemrograman

Deskripsi Project

Sistem Informasi Manajemen Parkir Tunai merupakan aplikasi berbasis Command Line Interface (CLI) yang dikembangkan menggunakan bahasa pemrograman Python sebagai proyek Ujian Akhir Semester Mata Kuliah Dasar Pemrograman.

Aplikasi ini dirancang untuk membantu proses pengelolaan parkir kendaraan secara sederhana, mulai dari pencatatan kendaraan masuk, kendaraan keluar, perhitungan biaya parkir, hingga penyusunan laporan parkir harian.

Fitur Aplikasi

Pencatatan Masuk

Menambahkan data kendaraan masuk

Pencatatan Keluar

Mengeluarkan kendaraan dan hitung biaya

Perhitungan Otomatis

Menghitung biaya parkir otomatis

Daftar Kendaraan

Menampilkan daftar kendaraan aktif

Laporan Harian

Menampilkan laporan parkir harian

Tampilan Rapi

Menggunakan PrettyTable untuk tabel

Validasi Input

Validasi input pengguna

Pembayaran

Sistem pembayaran Cash Only

Tampilan Program

Dashboard / Menu Utama

Daftar Kendaraan

Kendaraan Keluar

Laporan Parkir

Alur Program

Dashboard ↓ Kendaraan Masuk ↓ Daftar Kendaraan ↓ Kendaraan Keluar ↓ Daftar Keluar ↓ Laporan Parkir

Cara Menjalankan

1. Clone repository

git clone https://github.com/nico-alfianto/Parkir-Tunai-Python.git

2. Masuk ke folder project

cd Parkir-Tunai-Python

3. Install PrettyTable

pip install prettytable

4. Jalankan program

python AppParkir-1.py

Konsep Pemrograman yang Digunakan

  • Function - Modular Programming
  • List dan Dictionary - Penyimpanan Data
  • Looping dan Percabangan if-else
  • Input dan Output
  • Exception Handling
  • PrettyTable untuk tampilan tabel
